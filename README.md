# EDP : Analyse Numérique & Modélisation

On retrouvera dans ce répertoire différentes études consacrées à la résolution d'EDP classiques (Poisson, Chaleur, ...) par des méthodes de décomposition de domaine (Schwarz parallèle pour les équations stationnaires et SWR pour les instationnaires) en plusieurs dimensions (1D & 2D).
