# EDP : Analyse Numérique & Modélisation

On retrouvera dans ce répertoire différentes études consacrées à la résolution d'EDP classiques (Poisson, Chaleur, ...) en plusieurs dimensions (1D & 2D) et par des méthode de décompositions de domaines :
- Equations stationnaires : méthode de Schwarz parallèle (Lions)
- Equations instationnaires : méthode Schwarz Waveform Relaxation (SWR)
