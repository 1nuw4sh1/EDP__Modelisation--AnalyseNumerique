# Transport 1D

Equation de transport $\quad u_t + v \ u_x = 0 \quad$  sur $\quad \Omega = [0;1] \quad$ avec conditions de bords périodique.
Implémentation de divers schémas numériques :
- Spatial : Upwind, centré
- Temporel : Euler Explicite, RK 2-3-4
- Spatio-Temporel : Lax-Friedrichs, Lax-Wendroff
